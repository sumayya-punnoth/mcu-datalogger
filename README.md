# MCU Datalogger

4 layer PCB design of a datalogger using Atmega 328P-AU in KiCad.

<img width="650" height="419" alt="3D model of the PCB" src="https://github.com/user-attachments/assets/680ef817-765d-4ce9-9db2-92c328bfa3ba" />

## Viewing the project
1. Clone the repository:
```bash
git clone https://github.com/sumayya-punnoth/mcu-datalogger
```
2. Open `mcu_datalogger.kicad_pro` to view the schematic and PCB layout.
